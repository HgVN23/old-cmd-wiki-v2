Đối tượng có thể dùng Command
Player có OP

Command Block
Cú pháp
/function <name: filepath>
<name: filepath> là tên tệp tin có đuôi .mcfunction
Hướng dẫn tạo File Function
Trước tiên ấn Download để tải Pack mẫu về, sau đó giải nén và chuyển thư mục đó vào behavior_packs
B1: Mở thư mục function
B2: Tạo 1 tệp tin với tên bất kì và thêm đuôi .mcfunction
Lưu ý: nên ghi bình thường, không viết hoa, không cách (có thể dùng _ thay cho cách)
Ví dụ: give_stuff.mcfunction
B3: Mở tệp tin vừa tạo, ghi các Command. Ghi xong 1 Command thì xuống dòng để ghi Command tiếp theo. Sau khi xong nhớ ấn lưu
Trong Pack mẫu có 1 file tick.json là để chạy function thông qua Addon. Nó như là Command Block Reapeat Bạn có thể xóa tick.json hoặc xóa tên File function ghi trong tick.json nếu không muốn function luôn chạy thông qua Addon
Lưu ý
Nếu có quá nhiều tệp tin Function mà muốn phân biệt dễ dàng thì có thể tạo thư mục mới trong thư mục function để chứa các tệp tin đó
Ví dụ: tạo 1 thư mục mới tên abc trong thư mục functions và chuyển give_stuff.mcfunction vào abc thì khi /function bạn phải ghi:
/function abc/give_stuff
Có thể tạo thêm các thư mục khác trong các thư mục và khi /function chỉ cần ghi:
/function thu_muc_2/thu_muc_1/ten_tep_tin
