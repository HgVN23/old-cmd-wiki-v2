/execute
Công dụng
Khiến Entity thực hiện Command
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/execute <origin: target> <position: x y z> <command: command>
<origin: target> là mục tiêu (Entity)
<position: x y z> là tọa độ
<command: command> là Command
/execute @a ~~~ setblock ~~-1 ~ dirt — All Player sẽ Setblock Dirt dữ liệu 0 dưới chân
/execute <origin: target> <position: x y z> detect <detectPos: x y z> <block: Block> <data: int> <command: command>
<origin: target> là mục tiêu (Entity)
<position: x y z> là tọa độ
detect là kiểm tra
<detectPos: x y z> là tọa độ kiểm tra
<block: Block> là id Block
<data: int> là dữ liệu Block
<command: command> là Command
/execute @a ~~~ detect stone 0 setblock ~~-1 ~ dirt — All Player có Stone dữ liệu bằng 0 dưới chân sẽ bị Setblock Dirt dữ liệu 0 dưới chân, nếu không có thì không bị Setblock Dirt dưới chân
Lưu ý
<position: x y z> nên để mặc định là ~~~ cho mọi /execute
<block: Block> = -1 là mọi loại Block chung id
