/fog
Công dụng
Tạo hoặc xóa sương mù cho Player
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/fog <victim: target> push <fogId: string> <userProvideId: string>
<victim: target> là mục tiêu (Player)
push là đẩy, xuất hiện
<fogId: string> là id của Fog. Danh sách
<userProvideId: string> là set id Fog cho Player
/fog @a push minecraft:fog_soulsand_valley a — All Player sẽ có Fog của Soulsand Valley id là a
/fog <victim: target> <mode: delete> <userProvideId: string>
<victim: target> là mục tiêu (Player)
<mode: delete> là chế độ xóa. Danh sách:
pop là từ từ tan biến
delete là biến mất luôn
<userProvideId: string> là set id Fog cho Player
/fog @a pop a — All Player có Fog id là a sẽ từ từ tan biến
Lưu ý
1 Player có thể có nhiều Fog nhưng nó sẽ không hiện cùng lúc. Player này được Set Fog a trước sau đó là b thì cái b sẽ được hiện còn a sẽ bị ẩn
Fog có thể Set chung id nhưng khi xóa thì sẽ phải xóa nhiều lần. Player có Fog1 là a, Fog2 là a thì khi xóa a sẽ xóa cái Fog2 trước sau đó mới xóa được Fog1
