/tellraw
Công dụng
Gửi Chat dưới dạng mã tới mục tiêu xác định
Đối tượng có thể dùng Command
Player có OP
Chủ Map
Addon
/execute
/function
/gametest
Command Block
Cú pháp
/tellraw <target: target> <raw json message: json>
<target: target> là mục tiêu (Player)
<raw json message: json> là tin nhắn viết dưới dạng {"rawtext":[...]}. Danh sách
/tellraw @a {"rawtext":[{"text":"Hi"}]} — All Player nhận được Chat Hi
Lưu ý
Player có OP sẽ nhìn được Chat riêng
